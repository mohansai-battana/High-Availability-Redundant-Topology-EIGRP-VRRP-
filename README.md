# High-Availability-Redundant-Topology-EIGRP-VRRP-
Designed a topology with redundant links using EIGRP for fast-converging dynamic routing. Configured VRRP (Virtual Router Redundancy Protocol) to provide a virtual gateway, ensuring zero downtime for end-users during a router failure. Validated the failover process by capturing packets in Wireshark to confirm seamless traffic transition.
